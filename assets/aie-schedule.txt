# AI Engineer Europe 2026

> AI Engineer Europe is a 3-day conference bringing together the best AI engineers, researchers, and builders in Europe. Day 1 features hands-on workshops, Days 2-3 feature keynotes, talks across multiple tracks (AI Agents, Code, Open Source, Product, Infra), expo, and networking.

- Website: https://ai.engineer/europe
- Dates: April 8-10, 2026
- Location: London, UK
- Venue: Queen Elizabeth II Centre
- Tickets: https://app.ai.engineer/e/ai-engineer-europe-2026/portal
- Newsletter: https://ai.engineer/newsletter
- YouTube: https://youtube.com/@aidotengineer
- Twitter/X: https://x.com/aiDotEngineer
- LinkedIn: https://www.linkedin.com/company/aidotengineer/

## AIE for AI Engineers

Open endpoints for building apps, agents, and tools on conference data.

### Endpoints

- llms.txt (this file): https://ai.engineer/europe/llms.txt
- llms-full.txt (all sessions + speakers): https://ai.engineer/europe/llms-full.txt
- Sessions JSON: https://ai.engineer/europe/sessions.json
- Speakers JSON: https://ai.engineer/europe/speakers.json
- MCP Server: https://ai.engineer/europe/mcp
- iCal Calendar: https://ai.engineer/europe/calendar.ics
- Speaker Embeddings: https://ai.engineer/europe/speakers-embeddings.json
- Session Embeddings: https://ai.engineer/europe/sessions-embeddings.json
- Schedule Page: https://ai.engineer/europe/schedule

### MCP Server

The Model Context Protocol server lets AI agents (Antigravity, Codex, Claude, Cursor, etc.) query conference data with tool calls.

Tools: get_conference_info, list_speakers, list_sessions, get_schedule

Config (add to your MCP client):
```json
{
  "mcpServers": {
    "aie-europe": {
      "url": "https://ai.engineer/europe/mcp"
    }
  }
}
```

### Embeddings

Pre-computed Gemini Embedding 2 vectors for all speakers and sessions.
128-dim via Matryoshka Representation Learning (MRL) truncation from 3072.
Useful for semantic search, clustering, and recommendations.

- Speaker embeddings: https://ai.engineer/europe/speakers-embeddings.json
- Session embeddings: https://ai.engineer/europe/sessions-embeddings.json

### Agent Skills

Install the AI Engineer agent skill to give your coding agent full knowledge of the conference API:

  npx skills add aidotengineer/skills

Works with 40+ agents including Antigravity, Devin, Claude Code, Cursor, Windsurf, and Codex.
Source: https://github.com/aiDotEngineer/skills

### CLI Tool

Access conference data from your terminal (npm: @aidotengineer/aie):

  npx @aidotengineer/aie --list                  # list all conferences
  npx @aidotengineer/aie europe                  # Europe info (alias: eu)
  npx @aidotengineer/aie eu speakers             # speakers (page 1, 20/page)
  npx @aidotengineer/aie eu sessions --day help  # list all days
  npx @aidotengineer/aie eu search "agents"      # search sessions + speakers
  npx @aidotengineer/aie eu speakers --json      # raw JSON output

  curl https://ai.engineer/europe/llms.txt
